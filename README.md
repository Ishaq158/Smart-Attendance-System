The Smart Attendance System is a Java-based mini project developed to automate attendance marking using real-time face detection. The system captures live video from a camera, detects faces using OpenCV, and records attendance details such as date and time into a CSV file. This project demonstrates the application of Object-Oriented Programming concepts using Java along with basic computer vision techniques.
